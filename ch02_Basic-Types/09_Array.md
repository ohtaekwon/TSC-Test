### Array

- 원래 자바스크립트에서 **array** 는 객체이다.
- 사용방법
  - **Array<타입>**
  - 타입[]

```ts
// 배열을 표기하는 두 가지 방법.
// 1. union 타입으로 표기 방법.
let list:(number|string)[] = [1,2,3,"4"];

// 2. <> 을 표기하는 방법
let list:Array<number|string> = [1,2,3,"5"];
```
- 위의 코드에서 보통은 `2`의 방식보다 `1`의 방식을 더 많이 사용한다. 
- `2` 방식은 js, 나 ts에서 충돌이 날 수도 있다.
