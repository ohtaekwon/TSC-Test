# 09. Array

### 9.1. Array

- 원래 자바스크립트에서 **array** 는 객체이다.
- 사용방법
  - **Array<타입>**
  - 타입[]

### 9.2. Array 표기방법

_배열을 표기하는 방법으로 두 가지가 있다._

#### 9.2.1 union 표기방법

```ts
// 1. union 타입으로 표기 방법.
let list:(number|string)[] = [1,2,3,"4"];
```

#### 9.2.2. <> 표기방법

```ts
let list:Array<number|srting>=[1,2,3,4,"5"];
```

- 위의 코드에서 보통은 `<>`의 방식보다 `union`의 방식을 더 많이 사용한다. 
- `<>` 방식은 js, 나 ts에서 충돌이 날 수도 있다.
